## 🚀 快速启动

### 前置要求
- JDK 17+
- Node.js 18+
- Maven 3.8+
- MySQL 8.0+

### 启动步骤

```bash
# 1. 初始化数据库
mysql -uroot -p123456 < database/schema.sql
python database/init_db.py

# 2. 启动后端
cd backend
mvn spring-boot:run

# 3. 启动前端 (新终端)
cd web
npm install
npm run dev
```

### 访问地址
- 管理后台：http://localhost:3000
- 后端 API：http://localhost:8080
- 小程序：使用微信开发者工具打开 `mini-program/` 目录
