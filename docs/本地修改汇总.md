# 本地修改汇总（Web 端优化）

## 本次目标

- 关于我们页面改为 Markdown 展示，内容与小程序一致来源。
- 个人资料页改为用户友好头像更换流程（按钮选图，不再手输 URL）。
- 角色权限模块优化交互、支持角色状态修改、字段展示更易懂。
- 新增菜单管理模块（列表、增改删、状态管理）。
- 用户管理去掉 ID 列，优化列顺序与时间格式显示。
- 更新日志可直接看到实际更新内容（Markdown 预览）。
- 修复用户反馈回复无效问题。
- 首页样式与入口体验优化。
- 补充通用时间格式化与 Markdown 工具，修复多模块列表时间展示问题。

## 主要改动文件

### 后端

- `backend/src/main/java/com/ruobing/codebook/controller/RbacController.java`
  - 新增菜单保存/删除接口。
  - 新增菜单全量查询接口：`/rbac/menus/all`。
- `backend/src/main/java/com/ruobing/codebook/service/RbacService.java`
  - 新增菜单保存、删除、全量查询逻辑。
- `backend/src/main/java/com/ruobing/codebook/controller/FeedbackController.java`
  - 回复接口改为接收 `{ reply: "..." }` 结构化 JSON，避免字符串解析异常。

### 前端 API 与工具

- `web/src/api/index.js`
  - `feedbackApi.reply` 改为对象入参。
  - `rbacApi` 增加 `saveMenu`、`deleteMenu`、`getMenusAll`。
- `web/src/utils/format.js`（新增）
  - 提供 `formatDateTime` 与 `roleLabel`。
- `web/src/utils/markdown.js`（新增）
  - 提供 Markdown 到 HTML 转换。

### Web 页面

- `web/src/views/About.vue`
  - 改为 Markdown 预览 + 编辑双区模式。
- `web/src/views/Profile.vue`
  - 新增“更换头像”按钮和本地图片选择（Base64），保存后自动刷新资料。
- `web/src/views/Layout.vue`
  - 新增“菜单管理”导航入口。
  - 监听 `user-profile-updated` 事件，头像昵称实时刷新。
- `web/src/views/Rbac.vue`
  - 优化角色列表展示（角色名称、适用对象、状态开关）。
  - 支持编辑角色与状态即时保存。
  - 权限选择区样式与交互优化。
- `web/src/views/MenuManage.vue`（新增）
  - 菜单列表、新增、编辑、删除。
- `web/src/views/SysUser.vue`
  - 移除 ID 列，用户名调整为首列，时间格式化展示。
- `web/src/views/WebDesign.vue`
  - 去掉 ID 列，补充收藏列与格式化时间列。
- `web/src/views/Graduation.vue`
  - 去掉 ID 列，补充点赞/收藏列与格式化时间列。
- `web/src/views/Changelog.vue`
  - 新增“更新内容”列，Markdown 预览。
  - 时间格式统一。
- `web/src/views/Feedback.vue`
  - 去掉 ID 列，时间格式统一。
  - 回复前增加空值校验。
- `web/src/views/Home.vue`
  - 新增“常用入口”卡片区，优化管理后台首页访问效率与视觉层次。
- `web/src/router/index.js`
  - 注册菜单管理路由：`/menu-manage`。
